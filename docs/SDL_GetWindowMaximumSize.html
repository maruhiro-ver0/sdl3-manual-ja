SDL_GetWindowMaximumSize

ウィンドウのクライアント領域の最大サイズを得る

ヘッダ

SDL3/SDL_video.h

構文

bool SDL_GetWindowMaximumSize(SDL_Window *window, int *w, int *h);

引数

SDL_Window *window調査するウィンドウ
int *wウィンドウの最大幅の代入先へのポインタ. NULLも可
int *hウィンドウの最大高さの代入先へのポインタ. NULLも可

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

スレッドセーフ

この関数を呼べるのはメインスレッドのみである.

バージョン

SDL 3.2.0以降
