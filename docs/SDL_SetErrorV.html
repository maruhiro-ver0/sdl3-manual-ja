SDL_SetErrorV

現在のスレッドのSDLエラーメッセージを設定する

ヘッダ

SDL3/SDL_error.h

構文

bool SDL_SetErrorV(const char *fmt, va_list ap);

引数

const char *fmtprintf()形式の文字列形式
va_listap可変長引数リスト

戻り値

(bool) 常に偽を戻す.

詳細

この関数を呼ぶと, 以前のエラーメッセージは上書きされる.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
