SDL_GetWindowFromEvent

イベントに関連付けられたウィンドウを得る

ヘッダ

SDL3/SDL_events.h

構文

SDL_Window * SDL_GetWindowFromEvent(const SDL_Event *event);

引数

const SDL_Event *eventwindowIDを持つイベント

戻り値

(SDL_Window *) 関連付けられたウィンドウを戻す. 存在しないときNULLを戻す

スレッドセーフ

この関数はスレッドセーフである.

バージョン

SDL 3.2.0以降
