SDL_GetCurrentVideoDriver

初期化された現在のビデオドライバの名前を得る

ヘッダ

SDL3/SDL_video.h

構文

const char * SDL_GetCurrentVideoDriver(void);

戻り値

(const char *) 現在のビデオドライバの名前を戻す. 初期化されていないときNULLを戻す.

詳細

ドライバ名は全て"cocoa", "x11", "windows"のような単純な小文字のASCII文字の識別子である. Unicode文字が含まれることはない. また, 固有の名前として使うことを意図したものではない.

スレッドセーフ

この関数を呼べるのはメインスレッドのみである.
