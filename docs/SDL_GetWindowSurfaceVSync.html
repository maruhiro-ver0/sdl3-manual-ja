SDL_GetWindowSurfaceVSync

ウィンドウのサーフェイスの垂直更新周期を得る

ヘッダ

SDL3/SDL_video.h

構文

bool SDL_GetWindowSurfaceVSync(SDL_Window *window, int *vsync);

引数

SDL_Window *window問い合わせるウィンドウ
int *vsync現在の垂直更新周期の代入先へのポインタ. この値の詳細はSDL_SetWindowSurfaceVSync()を参照すること

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

スレッドセーフ

この関数を呼べるのはメインスレッドのみである.

バージョン

SDL 3.2.0以降
