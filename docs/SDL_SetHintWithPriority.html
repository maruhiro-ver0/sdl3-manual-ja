SDL_SetHintWithPriority

ヒントを指定の優先度で設定する

ヘッダ

SDL3/SDL_hints.h

構文

bool SDL_SetHintWithPriority(const char *name, const char *value, SDL_HintPriority priority);

引数

const char *name設定するヒント
const char *valueヒントの値
SDL_HintPrioritypriorityヒントの優先度のSDL_HintPriority

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

優先度はヒントの値が既に設定されている場合の振る舞いを決める. 既に設定されたヒントの優先度が同じか低ければ上書きされる. 環境変数の優先度は「上書き」である.

スレッドセーフ
