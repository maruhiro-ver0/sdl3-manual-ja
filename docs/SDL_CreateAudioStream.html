SDL_CreateAudioStream

オーディオストリームを生成する

ヘッダ

SDL3/SDL_audio.h

構文

SDL_AudioStream * SDL_CreateAudioStream(const SDL_AudioSpec *src_spec, const SDL_AudioSpec *dst_spec);

引数

const SDL_AudioSpec *src_specオーディオの入力形式
const SDL_AudioSpec *dst_specオーディオの出力形式

戻り値

(SDL_AudioStream *) 生成されたオーディオストリームを戻す. 失敗のときNULLを戻す. SDL_GetError()を呼んで詳細を知ることができる.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
