SDL_GetHintBoolean

ヒントの真偽値を得る

ヘッダ

SDL3/SDL_hints.h

構文

bool SDL_GetHintBoolean(const char *name, bool default_value);

引数

const char *name真偽値を求めるヒント
booldefault_valueヒントが存在しない場合に戻す値

戻り値

ヒントの真偽値を戻す. ヒントが存在しないときデフォルトの値を戻す.

この関数はどのスレッドからも安全に呼べる.
