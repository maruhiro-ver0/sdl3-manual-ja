SDL_HasProperty

プロパティグループにプロパティが存在するを得る

ヘッダ

SDL3/SDL_properties.h

構文

bool SDL_HasProperty(SDL_PropertiesID props, const char *name);

引数

SDL_PropertiesIDprops問い合わせるプロパティグループ
const char *name問い合わせるプロパティの名前

戻り値

(bool) プロパティが存在するとき真, 存在しないとき偽を戻す.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
