SDL_ClearError

エラーメッセージを消去する

ヘッダ

SDL3/SDL_error.h

構文

bool SDL_ClearError(void)

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す SDL_GetError()を呼んで詳細を知ることができる.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

サンプルコード

const char *error = SDL_GetError();
if (*error) {
  SDL_Log("SDLエラー: %s", error);
  SDL_ClearError();
}

バージョン

SDL 3.2.0以降
