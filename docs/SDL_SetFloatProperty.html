SDL_SetFloatProperty

プロパティグループの浮動小数点プロパティを設定する

ヘッダ

SDL3/SDL_properties.h

構文

bool SDL_SetFloatProperty(SDL_PropertiesID props, const char *name, float value);

引数

SDL_PropertiesIDprops設定するプロパティグループ
const char *name設定するプロパティの名前
floatvalueプロパティの新しい値

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

この関数はどのスレッドからも安全に呼べる.

バージョン
