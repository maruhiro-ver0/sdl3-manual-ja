SDL_GetSystemTheme

現在のシステムテーマを得る

ヘッダ

SDL3/SDL_video.h

構文

SDL_SystemTheme SDL_GetSystemTheme(void);

戻り値

(SDL_SystemTheme) 現在のシステムテーマ(ライト, ダーク)または不明を戻す. Returns the current system theme, light, dark, or unknown.

スレッドセーフ

この関数を呼べるのはメインスレッドのみである.

バージョン

SDL 3.2.0以降
