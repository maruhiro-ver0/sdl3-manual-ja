SDL_AddHintCallback

ヒントの値の変化の通知するために使用されるコールバックの型

ヘッダ

SDL3/SDL_hints.h

構文

typedef void(SDLCALL *SDL_HintCallback)(void *userdata, const char *name, const char *oldValue, const char *newValue);

引数

void *userdataSDL_AddHintCallback()のuserdataに渡したポインタ
const char *nameSDL_AddHintCallbackのnameに渡した名前
const char *oldValue変更前のヒントの値
const char *newValueこれから設定されるヒントの値

明細

コールバック関数は, SDL_AddHintCallbackが呼ばれたとき初期値を与えるために呼ばれ, さらにヒントの値が変わるたびに呼ばれる.

スレッドセーフ

このコールバックはヒントに新しい値が設定されたどのスレッドからも呼ばれる. SDLはこのスレッドを呼ぶときヒントサブシステムをロックする.

バージョン

SDL 3.2.0以降
