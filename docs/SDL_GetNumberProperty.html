SDL_GetNumberProperty

プロパティグループから数値プロパティを得る

ヘッダ

SDL3/SDL_properties.h

構文

Sint64 SDL_GetNumberProperty(SDL_PropertiesID props, const char *name, Sint64 default_value);

引数

SDL_PropertiesIDprops問い合わせるプロパティグループ
const char *name問い合わせるプロパティの名前
Sint64default_valueプロパティのデフォルト値

戻り値

(Sint64) プロパティの値を戻す. 値が設定されていない, または数値プロパティでない場合はdefault_valueを戻す.

詳細

SDL_GetPropertyType()でプロパティが存在し, かつ数値プロパティであるかを得られる.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
