SDL_main_func

アプリケーションのmain()関数のプロトタイプ型

ヘッダ

SDL3/SDL_main.h

構文

typedef int (SDLCALL *SDL_main_func)(int argc, char *argv[]);

引数

intargcANSI-Cスタイルのmain関数のargc
char **argvANSI-Cスタイルのmain関数のargv

戻り値

(int) ANSI-Cメインのreturn code;を戻す. 一般的に, 0のときプログラムが正常に終了したとみなされ, 0以外の小さな値のときエラーと見なされる.

バージョン

SDL 3.2.0以降
