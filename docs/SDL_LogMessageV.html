SDL_LogMessageV

指定の分類と重要度のログで出力する. これはSDL_LogMessage()の可変引数リスト版である.

ヘッダ

SDL3/SDL_log.h

構文

void SDL_LogMessageV(int category, SDL_LogPriority priority, const char* fmt, va_list ap);

引数

intcategoryメッセージの分類
SDL_LogPrioritypropertyメッセージの重要度
const char *fmtprintf()形式の文字列形式
va_listap可変引数リスト
