SDL_Locale

ユーザの設定したロケールの構造体

バージョン

SDL3/SDL_locale.h

フィールド

const char *language言語, 例えば英語ならば"en"
const char *country国, 例えばアメリカ合衆国ならば"US", NULLもありうる

詳細

ロケールデータは, 言語(例えば英語のような)と, 国(例えばカナダのような, ない場合もある)に分けられる. languageはISO-639形式(英語ならば"en")で, countryはNULLでなければISO-3166国名コード(カナダならば"CA")である.

バージョン

SDL 3.2.0以降
