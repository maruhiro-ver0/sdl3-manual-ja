SDL_GetWindowSize

ウィンドウのクライアント領域のサイズを得る

ヘッダ

SDL3/SDL_video.h

構文

bool SDL_GetWindowSize(SDL_Window *window, int *w, int *h);

引数

SDL_Window *window調査するウィンドウ
int *wウィンドウの幅の代入先へのポインタ. NULLも可
int *hウィンドウの高さの代入先へのポインタ. NULLも可

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

ウィンドウが高DPIディスプレイ上にある場合, ウィンドウのピクセルサイズはウィンドウ座標系のサイズとは異なる場合がある. SDL_GetWindowSizeInPixels()またはSDL_GetRenderOutputSize()で実際のピクセル数でクライアントサイズを得ることができる.

スレッドセーフ

この関数を呼べるのはメインスレッドのみである.

バージョン

SDL 3.2.0以降
